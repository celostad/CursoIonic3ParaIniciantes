A Documentação de componentes Ionic apresenta vários componentes úteis que são incluídos com o Ionic.

O que é Ionic

O Ionic Framework é um SDK de software livre que permite aos desenvolvedores criar aplicativos móveis de alta qualidade com tecnologias da Web, como HTML, CSS e JavaScript.

Oque é Ionic Cloud

O Ionic Cloud é uma plataforma na nuvem para gerenciar e dimensionar aplicativos iônicos com serviços integrados, como notificações por push, compilações nativas, autenticação de usuário e atualização ao vivo.
